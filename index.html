Olvix license services

This host publishes the signed Olvix enterprise-license revocation list. The only artifact that matters here is the signed document at /revocations.jwt.

It is an Ed25519-signed JWT — integrity comes from the signature, not from this host. Self-hosted Olvix installs verify it against the embedded license public key; a missing or unverifiable file fails open (nothing revoked). Community installs never fetch it.

Operated by Industrious Hive.
